# utshell

#### 描述

GNU Bourne Again shell （Bash） 是与 Bourne shell （sh） 兼容的 shell 或命令语言解释器。Bash 整合了 Korn shell （ksh） 和 C shell （csh） 的有用功能。大多数 sh 脚本都可以由 utshell 运行，无需修改。
